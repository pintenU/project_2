Grunderna

HTML-element

Meta

<!DOCTYPE> Denna deklaration specificerar vilken version av HTML som används i dokumentet.

<html> Rotelementet i ett HTML-dokument som innehåller alla andra element.

<head> En behållare för metadata om dokumentet, såsom titeln, länkar till stilmallar och metataggar.

<title> Definierar titeln på HTML-dokumentet, som visas i webbläsarens titelrad eller flik.

<style> Används för att bädda in CSS (Cascading Style Sheets) i ett HTML-dokument.

<link> Specificerar externa resurser, som CSS-filer, att användas i dokumentet.

<body> Huvudinnehållsområdet i HTML-dokumentet där text, bilder och annat innehåll visas.

Semantiska element

<main> Representerar huvudinnehållet i dokumentet, och det skall endast finnas ett <main>-element på varje HTML-sida.

<header> Innehåller vanligtvis introducerande innehåll eller en webbplatslogotyp och navigation som rör hela webbplatsen.

<section> Delar upp innehållet i avsnitt, ofta med en rubrik, och används för att gruppera relaterat innehåll.

<footer> Innehåller vanligtvis information om företaget/skaparen, upphovsrätt eller kontaktuppgifter.

<article> Representerar en fristående del av innehåll som kan distribueras och återanvändas oberoende.

<aside> Innehåller innehåll som är delvis relaterat till huvudinnehållet, ofta används för sidofält.

Text & Rubriker

<a> Skapar hyperlänkar till andra webbsidor eller resurser.

<span> En generisk behållare som används för att applicera stilar eller skript på en specifik del av texten.

<p> Representerar ett stycke text.

<h1> - <h6> Rubriker (h1 är störst).

<ol> Ordnad lista.

<ul> Oordnad lista.

<li> Listobjekt.

<strong> Fet stil (viktigt för SEO).

<em> Kursiv stil.

<div> Generisk behållare.
